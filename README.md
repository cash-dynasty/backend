# Cash Dynasty

## Prerequisites
- Python 3.10+

## Configuration
1. Run commands:
```bash
# create a virtual environment
python -m venv venv

# activate the virtual environment
source venv/bin/activate

# install required dependencies and configure pre-commit hooks
make configure
```
2. Fill in the .env file.

## Running
```
make start
```

## Linting
```
make lint
```

## Testing
```
make test
```
